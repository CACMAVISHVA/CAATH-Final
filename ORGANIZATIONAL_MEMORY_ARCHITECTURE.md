# Organizational Memory Architecture

CAATH now includes an organizational memory layer for institutional operational intelligence.

This is not generic document storage, a passive wiki, or a disconnected knowledge base. Memory records are workflow-linked, explainable, governed, and surfaced inside operational workspaces.

## Locations

- Memory domain: `src/domains/organizational-memory/`
- Learning dashboard: `src/domains/learning-dashboard/`
- Workspace snapshot: `KnowledgeSnapshot.tsx`
- Command integration: `src/services/commandPaletteService.ts`

## Capabilities

- Institutional memory records.
- Workflow outcome memory.
- Escalation resolution memory.
- Operational pattern retention.
- Resolution lineage.
- Confidence indicators.
- Knowledge governance controls.
- Runtime-safe pruning and confidence recalibration.
