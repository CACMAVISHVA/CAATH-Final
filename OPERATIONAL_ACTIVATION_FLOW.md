# Operational Activation Flow

Date: 2026-05-25

## Flow
1. User enters workspace.
2. Role-aware activation modal opens (if not completed).
3. User completes contextual setup/workflow/tour milestones.
4. Workspace template is provisioned.
5. User is guided into first operational actions.
6. Progress and telemetry are captured continuously.

## Activation Milestones
- first setup action
- first workflow completion
- first guided tour milestone
- workspace provisioning applied
- activation completion

## UX Principles
- progressive disclosure
- skip/resume support
- role-aware context
- operationally relevant tasks first

## Integration Points
- Workspace preferences for quick pins
- Tab navigation for direct workflow entry
- Telemetry for adoption intelligence
