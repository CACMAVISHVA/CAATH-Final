# CAATH Operational Spacing System

CAATH preserves information density through structure, not compression.

## Density Modes

- Compact: fast scanning, smaller rows, slightly tighter cards.
- Standard: default operator mode with balanced padding and readable row rhythm.
- Executive: more generous spacing and larger type for monitoring surfaces.

## Layout Rules

- Shell padding should generally start at `p-5` and scale to `p-7` on large screens.
- Primary grids use `gap-5` where possible.
- Related controls use `gap-2`; unrelated controls need more separation.
- Panel interiors should use `p-5` as the default.
- Tables may remain dense, but headers and selected states should have breathing room.

## Reducing Noise

When a region feels crowded, remove borders first, then reduce badge count, then group by spacing. Do not remove essential operational data unless the workflow no longer needs it.
