# CAATH Premium Enterprise Navigation

Premium enterprise navigation should feel calm, predictable, and role-aware.

## Sidebar Rules

- Dashboard appears first.
- Live Workspace appears second.
- Section labels are scannable and compact.
- Active states should be clear without glowing or heavy outlines.
- Icons align to a stable column and labels truncate cleanly.
- Pinned and recent items sit below primary navigation.

## Role-Aware Behavior

Roles may hide restricted capabilities, but visible items should retain the same hierarchy. Administrative and workflow utility items should sit below the main operating path.

## Evolution Direction

Future navigation refinements should add contextual shortcuts, favorites, and recent entities without disrupting the core section model.
