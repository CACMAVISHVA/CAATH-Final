# Enterprise Accountability Framework

The accountability framework makes operational responsibility explicit.

## Accountability Signals

- Current owner.
- Workflow state.
- SLA timer.
- Approval checkpoint.
- Escalation chain.
- Handoff status.
- Operator action attribution.

## Governance Dashboard

The Accountability Surface in `GovernanceDashboard.tsx` shows:

- Workflow name.
- Owner.
- State.
- SLA status.

## Outcome

Operators and leaders can see who owns each workflow, what the next governed action is, and where responsibility changes occurred.
