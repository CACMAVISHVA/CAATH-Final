# Multi-Panel Workflow System

The workspace shell supports side-by-side workflows for enterprise multitasking.

## Layouts

- `triage`: balanced task, workflow, and awareness panels.
- `execution`: emphasizes workflow execution and side-by-side work rooms.
- `executive`: emphasizes live metrics and operational visibility.

## Panel Types

- Live tasks.
- Notices.
- Intelligence.
- Escalations.
- Activity.
- Alerts.
- Collaboration.
- Timeline.

## Persistence

Workspace state persists:

- Active layout.
- Active dock panel.
- Open workspace tabs.
- Pinned views.
- Recent sessions.
- Focus mode.
- Density mode.
