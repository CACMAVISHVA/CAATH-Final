# Executive Coordination Infrastructure

## Cognitive Command Center
The `cognitive-command-center` domain exposes live execution visibility for executive and operations roles:
- objective tracking score
- cognitive routing heatmaps
- strategic action cards
- governance/override visibility
- cognitive operational timeline

## Coordination Principles
- execution-focused, not dashboard-only insights
- permission-aware action routing
- reviewable strategic handoff lifecycle
- ergonomic workflow-first operator UX
