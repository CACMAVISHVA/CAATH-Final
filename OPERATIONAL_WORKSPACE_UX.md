# Operational Workspace UX

The Realtime Workspace is designed for operators who need continuous context while handling live work.

## UX Principles

- Keep work visible while switching panels.
- Preserve session state between visits.
- Make live operational movement visible.
- Put actions close to the work item.
- Keep AI suggestions contextual and operator-owned.
- Favor compact, high-density execution layouts.

## Primary Surfaces

- Live task panel.
- Realtime notice panel.
- Docked intelligence panel.
- Escalation and alert panels.
- Collaboration panel.
- Unified activity timeline.
- Persistent pinned views and recent sessions.

## Focus Mode

Focus mode reduces the workspace to a compact execution view with rapid actions and the most relevant side context.
