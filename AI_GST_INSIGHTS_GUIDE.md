# AI GST Insights Guide

Date: 2026-05-25

## AI Layer Scope
The GST AI layer provides:
- compliance summaries
- risk prioritization
- workflow acceleration suggestions

It does not auto-file returns or execute irreversible actions.

## Governance Controls
- AI output is advisory.
- Every insight contains governance notes.
- Role-aware usage is enforced by existing workspace identity model.
- Intelligence execution is telemetry-traced for audit visibility.

## Insight Patterns
- Elevated audit exposure
- Vendor inconsistency risk
- Workflow acceleration recommendation

## Safe Operations Model
1. AI generates ranked recommendations.
2. Authorized operator validates source evidence.
3. Operator triggers workflows/tasks/escalations.
4. Actions are logged via runtime telemetry/audit systems.

## Future Upgrades
- Confidence calibration per module.
- Role-specific narrative depth (Staff vs Admin vs Executive).
- Prompt policy registry integration for stricter AI controls.
