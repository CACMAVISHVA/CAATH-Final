# CAATH Enterprise Dark Theme Guide

The CAATH dark theme uses graphite depth rather than pure black contrast.

## Theme Tokens

- Base background: `#15171c`
- Primary surface: `#1d2128`
- Muted gold: `#c8a85a`
- Gold light: `#e3cb86`
- Gold dark: `#8a7240`

## Usage Rules

- Avoid pure black backgrounds except for intentional depth anchors.
- Use translucent white overlays for nested surfaces.
- Use rings below `0.07` opacity for most container definition.
- Use semantic colors at low opacity for awareness, not decoration.
- Avoid neon glows, bright outlines, and saturated gradients.

## Scrollbars And Chrome

Scrollbars should be neutral by default and only warm on hover. Navigation chrome should frame the workspace quietly instead of competing with operational content.
