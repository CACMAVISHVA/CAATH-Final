# Historical Intelligence Architecture

## Domain Layers
- `compliance-knowledge`: knowledge graph + pattern intelligence + memory
- `compliance-history-dashboard`: trend/dashboard projection model

## Execution Path
1. GST execution result generated
2. Resolution center output generated
3. Knowledge graph orchestrator builds entities/relations/patterns
4. History dashboard orchestrator projects executive historical views

## Design Principles
- explainable comparisons
- lineage-linked historical reasoning
- cross-domain operational correlation
- governed, permission-aware intelligence outputs
