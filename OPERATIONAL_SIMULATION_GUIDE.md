# Operational Simulation Guide

Date: 2026-05-25

## Goal
Support safe what-if planning before executing high-impact workflow changes.

## Simulation Inputs
- scenario type (`reassignment`, `escalation`, `staffing`, `redistribution`)
- delta workflows
- current overdue count
- current escalation count
- available staff baseline

## Simulation Outputs
- projected overdue volume
- projected escalations
- projected SLA risk
- confidence score
- explanation lines

## Planning Usage
Use simulation previews in operations center before triggering major redistribution/escalation actions.
