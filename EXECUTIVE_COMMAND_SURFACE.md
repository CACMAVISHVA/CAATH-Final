# CAATH Executive Command Surface

The executive command surface is the top operational control area for workspace identity, live state, command access, and rapid execution.

## Required Elements

- Workspace identity and live sync state.
- Command palette entry with `Ctrl K` visibility.
- Focused mode selector for triage, execution, and executive monitoring.
- Focus toggle for deep operational work.
- Session tabs and rapid queue traversal.
- One clear primary quick action.

## Design Rules

- Keep command controls in one unified surface.
- Avoid stacked mini toolbars with equal visual weight.
- Use segmented controls for modes and restrained buttons for secondary actions.
- Reserve the primary action style for the dominant operational action.

## Speed Principle

Visual simplification must not reduce speed. Keyboard access, command execution, and queue traversal remain visible and immediate.
