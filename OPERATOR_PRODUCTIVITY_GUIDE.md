# Operator Productivity Guide

CAATH operators should be able to move from awareness to action without hunting through modules.

## Primary Shortcuts

- `Ctrl+K`: Open universal search and command palette.
- `Alt+N`: Create task.
- `Alt+A`: Open approvals.
- `Alt+W`: Trigger workflow.
- `Alt+Q`: Quick approve.
- `Alt+R`: Bulk resolve.

## Daily Operating Model

1. Start in Command Center.
2. Review SLA health, priority inbox, approvals, escalations, and workload.
3. Use the work queue table for selection, bulk assignment, and quick approvals.
4. Use pinned and recent entities for repeated client or workflow access.
5. Use Ctrl+K for instant navigation across clients, GSTINs, tasks, notices, documents, approvals, and workflows.

## Productivity Layers

- Personal cockpit: My Tasks, My Notices, My Clients, My Approvals, My Alerts, My Intelligence.
- Workflow acceleration: bulk actions, mass assignment, quick approvals, workflow shortcuts, one-click resolution affordances.
- Notification center: grouped, priority-scored, action-ready notifications.
- Activity stream: operational timeline across workflow, intelligence, escalation, and approval events.
