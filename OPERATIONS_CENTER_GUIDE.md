# Operations Center Guide

## Implemented Snapshot
- `OperationsCenterOrchestrator` combines:
  - runtime health (queue/realtime/notification/security)
  - operational analytics summary
  - AI operational recommendations

## Command Center Alignment
- Keyboard-first command shortcuts integrated via command-center hook.
- Command actions remain role-aware and runtime-observable.

## Target UX
- Single-pane operational command visibility.
- Realtime execution and risk signals in one operator context.
- Workflow-centric routing over menu-centric navigation.
