# Compliance Knowledge Graph

## Overview
`src/domains/compliance-knowledge/` provides enterprise compliance memory through linked entities and relationships across GST operations.

## Tracked Entities
- clients
- vendors
- filings
- reconciliation clusters
- workflows
- audit risk nodes
- resolution nodes

## Relationship Model
- filing-to-client lineage
- reconciliation-to-resolution linkage
- vendor-risk dependency edges
- audit-risk correlation edges

## Outcomes
- operational memory graph
- recurring risk pattern intelligence
- cross-domain correlation signals
- explainable historical narratives
