# CAATH Operational Visual Grouping

Operational grouping turns a large enterprise workspace into readable zones.

## Group Types

- Overview: business state and high-level operational posture.
- Operations: active work execution.
- Collaboration: shared work, mentions, handoffs, and command-center awareness.
- Control: governance, auditability, approvals, and accountability.
- Knowledge: institutional memory, playbooks, and learning surfaces.

## Composition Rules

- Use whitespace before separators.
- Use subtle surfaces before boxed cards.
- Keep related controls in clusters.
- Keep context panels visibly secondary to workflow execution.
- Avoid duplicating the same action in multiple nearby toolbars.

## Panel Rhythm

Primary panels can carry stronger elevation. Secondary panels should use the same component system with quieter tone, smaller emphasis, and fewer attention markers.
