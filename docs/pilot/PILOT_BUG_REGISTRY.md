# CAATH Pilot Bug Registry

This registry tracks defects and operational blockers discovered during internal pilot use at AVA Associates. Record only issues observed during real workflows.

## Severity Classification

| Severity | Definition |
| --- | --- |
| P0 | Blocks daily operations |
| P1 | Major workflow friction |
| P2 | Minor usability issue |
| P3 | Cosmetic improvement |

## Fix Status

| Status | Definition |
| --- | --- |
| Open | Bug has been observed and is awaiting investigation or fix |
| Investigating | Root cause is being identified |
| In Progress | Fix is actively being worked on |
| Fixed | Fix has been implemented |
| Verified | Fix has been validated in real daily use |
| Deferred | Not being addressed during the current pilot cycle |
