# CAATH Feature Retirement Candidates

Use this list to identify features, navigation items, and workflows that should be merged, simplified, hidden, or removed based on real pilot usage.

## Retirement Criteria

| Criterion | Description |
| --- | --- |
| Never used | Feature or navigation item was available but not used during pilot operations |
| Duplicate capability | Feature overlaps with another workflow or module without adding clear value |
| Unnecessary complexity | Feature creates confusion, extra steps, or unclear decisions |
| Merge candidate | Feature has value but should be folded into a stronger existing workflow |
| Remove candidate | Feature adds little value and creates maintenance or usability burden |
