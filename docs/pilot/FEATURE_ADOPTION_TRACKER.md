# CAATH Feature Adoption Tracker

Use this tracker to observe how AVA Associates actually uses CAATH during internal pilot operations. The goal is to identify features to improve, merge, simplify, or remove.

## Tracking Principles

- Track actual daily usage, not intended usage.
- Treat unused features as evidence, not failure.
- Watch for repeated manual actions that should be easier.
- Record confusing workflows even when users eventually complete them.
- Do not add new architecture or new feature domains from this tracker.
