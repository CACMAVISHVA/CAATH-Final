# Keyboard-First Operation System

CAATH supports a keyboard-first operational model.

## Shortcuts

- `Ctrl+K`: Global command palette.
- `Alt+X`: Open realtime workspace.
- `Alt+F`: Deep-work mode.
- `Alt+T`: Rapid triage layout.
- `Alt+M`: Executive monitoring layout.
- `Alt+N`: Create task.
- `Alt+A`: Open approvals.
- `Alt+Q`: Quick approve.
- `Alt+R`: Bulk resolve.
- `Alt+W`: Trigger workflow.
- `J`: Next queue item.
- `K`: Previous queue item.
- `Alt+1` to `Alt+8`: Dock panel switching.

## Design Goal

Every high-frequency operational move should have a command, a hotkey, or an inline action close to the workflow.
